This folder is a minimal public website package for the course assignments.

Files:
- index.html: homepage built from the Week 1 slide submission
- hw1/index.html: HW1 blog-style post for A/B testing and frequentist inference

To publish on GitHub Pages:
1. Create a public repository on GitHub.
2. Upload all files in this folder.
3. In Settings > Pages, set Source to Deploy from a branch, Branch to main, Folder to / (root).
4. Your website URL will be https://USERNAME.github.io/REPO/
5. Your HW1 post URL will be https://USERNAME.github.io/REPO/hw1/
